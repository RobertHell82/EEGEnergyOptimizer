**Method 1: `_renderEnergyFlow()`**

Creates an inline SVG (~300x250) with 4 labeled nodes positioned as:
- PV: top center (icon: sun or solar-panel shape)
- Battery: left (icon: battery shape)
- House: right (icon: house shape)
- Grid: bottom center (icon: power-plug shape)

Each node is a rounded rect or circle with label + current value text.

Read sensor values at the top of the method:
```js
const pvW = this._readFloat(this._config?.pv_power_sensor) || 0;
const batW = this._readFloat(this._config?.battery_power_sensor) || 0; // +charge, -discharge
const gridW = this._readFloat(this._config?.grid_power_sensor) || 0;  // +import, -export
const socVal = this._readFloat(this._config?.battery_soc_sensor);
// House consumption from EEG sensor (kW, convert to W for display consistency)
const hausState = this._readFloat("sensor.eeg_energy_optimizer_hausverbrauch");
const hausW = hausState != null ? hausState * 1000 : Math.max(0, pvW - batW - gridW);
```

Calculate flows between nodes:
- PV -> House: always `min(pvW, hausW)` when pvW > 0
- PV -> Battery: `min(pvW - hausW, batW)` when pvW > hausW AND batW > 0 (charging)
- PV -> Grid: `Math.abs(gridW)` when gridW < 0 (export) AND pvW > 0
- Battery -> House: `Math.abs(batW)` when batW < 0 (discharging)
- Grid -> House: `gridW` when gridW > 0 (import)

For each flow > 0, draw a path line between the two nodes:
- Stroke width: `Math.max(1.5, Math.min(6, flow / 500))` (scale 0-3000W to 1.5-6px)
- Color: green (#4CAF50) for PV/export paths, orange (#FF9800) for battery paths, red (#f44336) for grid import
- CSS animation: `stroke-dasharray: 8 4; animation: flowDash 1s linear infinite;` for active flows
- Direction: `stroke-dashoffset` animation direction indicates flow direction

Node value display:
- PV node: `${(pvW/1000).toFixed(1)} kW` in green
- Battery node: `${Math.abs(batW/1000).toFixed(1)} kW` + `${socVal != null ? Math.round(socVal) + '%' : ''}` — orange text
- House node: `${(hausW/1000).toFixed(1)} kW` in blue (#2196F3)
- Grid node: `${(Math.abs(gridW)/1000).toFixed(1)} kW` — green if export (gridW < 0), red if import (gridW > 0)

Add `@keyframes flowDash { to { stroke-dashoffset: -12; } }` to the CSS section. For reverse direction flows, use `stroke-dashoffset: 12` and animate `to { stroke-dashoffset: 0; }` — or use a second keyframe `flowDashReverse`.
