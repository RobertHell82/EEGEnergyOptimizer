**2. Create optimizer.py** with:

- `Snapshot` dataclass: fields `now` (datetime), `battery_soc` (float, default 0.0), `battery_capacity_kwh` (float, default 0.0), `pv_remaining_today_kwh` (float|None, default None), `pv_tomorrow_kwh` (float|None, default None), `consumption_today_kwh` (float, default 0.0), `consumption_tomorrow_kwh` (float, default 0.0), `consumption_to_sunrise_kwh` (float, default 0.0), `sunrise` (datetime|None, default None), `sunset` (datetime|None, default None)

- `Decision` dataclass: fields `timestamp` (str, ""), `zustand` (str, "Normal"), `ueberschuss_faktor` (float, 0.0), `ladung_blockiert` (bool, False), `entladung_aktiv` (bool, False), `entladeleistung_kw` (float, 0.0), `min_soc_berechnet` (float, 0.0), `naechste_aktion` (str, ""), `markdown` (str, ""), `ausfuehrung` (bool, False), `block_reasons` (list[str], field(default_factory=list))

- `EEGOptimizer` class:
  - `__init__(self, hass, config, inverter, coordinator, provider)` -- stores references, reads config values: `_ueberschuss_schwelle` from `CONF_UEBERSCHUSS_SCHWELLE` (default 1.25), `_morning_end_hour/min` parsed from `CONF_MORNING_END_TIME` (default "10:00"), `_discharge_start_h/m` parsed from `CONF_DISCHARGE_START_TIME` (default "20:00"), `_discharge_power_kw` from `CONF_DISCHARGE_POWER_KW` (default 3.0), `_min_soc` from `CONF_MIN_SOC` (default 10), `_safety_buffer_pct` from `CONF_SAFETY_BUFFER_PCT` (default 25). Also init `_prev_zustand = None` for inverter deduplication.
  - `_gather_snapshot(self) -> Snapshot`: reads battery SOC from `_read_float(hass, config[CONF_BATTERY_SOC_SENSOR])`, capacity from sensor or manual fallback (same logic as BatteryMissingEnergySensor._resolve_capacity), PV from `self._provider.get_forecast()`, consumption today/tomorrow from `self._coordinator.calculate_period()`, consumption to sunrise from `self._coordinator.calculate_period(now, sunrise)`. Gets sunrise/sunset from `hass.states.get("sun.sun")` using `next_rising` attribute. When sun is above horizon, today's sunrise = `next_rising - 24h`. Tomorrow's sunrise = `next_rising` when sun above horizon, else `next_rising`.
  - `_calc_ueberschuss_faktor(self, snap) -> float`: `pv_remaining_today_kwh / consumption_today_kwh`. Returns 0.0 if pv is None or <= 0. Returns 99.0 if consumption <= 0.
  - `_should_block_charging(self, snap) -> bool`: per D-01 to D-04. Only on surplus days (factor >= threshold, D-03). Time window: `sunrise - 1h` to `morning_end_time` (D-01, D-02). Returns False if sunrise is None.
  - `_calc_min_soc(self, snap) -> float`: per D-08. Formula: `self._min_soc + ceil((overnight_kwh * (1 + self._safety_buffer_pct/100)) / capacity * 100)`. Returns `self._min_soc` if capacity <= 0.
  - `_should_discharge(self, snap) -> tuple[bool, float, list[str]]`: per D-05 to D-09. Checks: (1) time >= discharge start, (2) soc > min_soc, (3) pv_tomorrow >= tomorrow_demand where tomorrow_demand = consumption_tomorrow + (100 - base_min_soc)/100 * capacity (battery charge needed). Returns (should_discharge, min_soc, block_reasons).
  - `_evaluate(self, snap, mode) -> Decision`: orchestrates the above. Sets `zustand` to one of STATE_MORGEN_EINSPEISUNG/STATE_NORMAL/STATE_ABEND_ENTLADUNG (D-22). Sets `ausfuehrung = (mode == MODE_EIN)` (D-17). Populates `naechste_aktion` with German text like "Morgen-Einspeisung bis 10:00" or "Abend-Entladung 20:00" or "Normalbetrieb" (D-25). Calls `_build_markdown()` for the markdown attribute.
  - `_build_markdown(self, snap, decision) -> str`: builds Markdown text per D-26 to D-29. Contains current status, next action, and either morning block info (D-29) or evening discharge info (D-28) depending on time of day. Uses `\n` newlines. Format example for morning: `## Status\nMorgen-Einspeisung\n\n### Ladung blockiert\n- Blockiert bis: 10:00\n- PV Prognose heute: 25.3 kWh\n...`
  - `async _execute(self, decision, snap)`: only called when `decision.ausfuehrung is True`. Deduplicates against `_prev_zustand`: if `decision.zustand == self._prev_zustand`, skip inverter calls. For Morgen-Einspeisung: `inverter.async_set_charge_limit(0)`. For Abend-Entladung: `inverter.async_set_discharge(power_kw, target_soc=min_soc)`. For Normal: `inverter.async_stop_forcible()`. Update `_prev_zustand` after successful call.
  - `async async_run_cycle(self, mode) -> Decision`: calls `_gather_snapshot()`, `_evaluate(snap, mode)`, optionally `_execute(decision, snap)` if mode == MODE_EIN. Returns the Decision (for sensor updates). Wraps everything in try/except to log errors and return a fallback Decision.
  - Property `last_decision -> Decision | None` for sensor access.

```python
"""Select platform for EEG Energy Optimizer."""
from __future__ import annotations
import logging
from typing import TYPE_CHECKING, Any

from .const import DOMAIN, MODE_AUS, MODE_EIN, MODE_TEST, OPTIMIZER_MODES

if TYPE_CHECKING:
    from homeassistant.config_entries import ConfigEntry
    from homeassistant.core import HomeAssistant
    from homeassistant.helpers.entity_platform import AddEntitiesCallback

_LOGGER = logging.getLogger(__name__)

# HA imports guarded for test environment
try:
    from homeassistant.components.select import SelectEntity
    from homeassistant.helpers.device_registry import DeviceEntryType
    from homeassistant.helpers.entity import DeviceInfo
    from homeassistant.helpers.restore_state import RestoreEntity
except ImportError:
    class SelectEntity:  # type: ignore[no-redef]
        _attr_has_entity_name: bool = False
        _attr_name: str = ""
        _attr_unique_id: str = ""
        _attr_options: list[str] = []
        _attr_current_option: str | None = None
        _attr_device_info: Any = None
        _attr_icon: str | None = None
        async def async_select_option(self, option: str) -> None: pass
        def async_write_ha_state(self) -> None: pass
    class RestoreEntity:  # type: ignore[no-redef]
        async def async_get_last_state(self): return None
    class DeviceEntryType:  # type: ignore[no-redef]
        SERVICE = "service"
    class DeviceInfo(dict):  # type: ignore[no-redef]
        def __init__(self, **kwargs): super().__init__(**kwargs)


class OptimizerModeSelect(SelectEntity, RestoreEntity):
    """Select entity for optimizer mode: Ein / Test / Aus."""

    _attr_has_entity_name = True
    _attr_name = "Optimizer"
    _attr_icon = "mdi:robot"
    _attr_options = OPTIMIZER_MODES
    _attr_current_option = MODE_AUS

    def __init__(self, entry_id: str) -> None:
        self._attr_unique_id = f"{DOMAIN}_{entry_id}_optimizer"
        self._attr_device_info = DeviceInfo(
            identifiers={(DOMAIN, entry_id)},
            name="EEG Energy Optimizer",
            manufacturer="Custom",
            model="EEG Energy Optimizer",
            entry_type=DeviceEntryType.SERVICE,
        )

    async def async_added_to_hass(self) -> None:
        last_state = await self.async_get_last_state()
        if last_state and last_state.state in self._attr_options:
            self._attr_current_option = last_state.state
            _LOGGER.info("Restored optimizer mode: %s", last_state.state)

    async def async_select_option(self, option: str) -> None:
        self._attr_current_option = option
        self.async_write_ha_state()
        _LOGGER.info("Optimizer mode changed to: %s", option)


async def async_setup_entry(
    hass: HomeAssistant,
    entry: ConfigEntry,
    async_add_entities: AddEntitiesCallback,
) -> None:
    select = OptimizerModeSelect(entry.entry_id)
    hass.data[DOMAIN][entry.entry_id]["select"] = select
    async_add_entities([select])
```
