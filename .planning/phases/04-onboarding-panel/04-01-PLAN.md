**1. Create `websocket_api.py`** (NEW file) with these 4 WebSocket command handlers:

```python
"""WebSocket API for EEG Optimizer panel."""
```

- **`eeg_optimizer/get_config`** — No extra params. Looks up `hass.config_entries.async_entries(DOMAIN)`, takes first entry, returns `{**entry.data, **entry.options}`. If no entry found, return error `"not_configured"`.

- **`eeg_optimizer/save_config`** — Requires `vol.Required("config"): dict`. Looks up first config entry. Calls `hass.config_entries.async_update_entry(entry, data={**entry.data, **msg["config"]})`. Returns `{"success": True}`. If no entry, error `"not_configured"`.

- **`eeg_optimizer/check_prerequisites`** — No extra params. Checks `hass.config_entries.async_entries(domain)` for domains: `huawei_solar`, `solcast_solar`, `forecast_solar`. For each, check if any entry has `e.state.value == "loaded"`. Returns dict like `{"huawei_solar": true, "solcast_solar": false, "forecast_solar": true}`.

- **`eeg_optimizer/detect_sensors`** — No extra params. Uses the same logic as `config_flow.py`'s `_find_huawei_battery_device()` and `HUAWEI_DEFAULTS` to auto-detect Huawei sensors. Import `HUAWEI_DEFAULTS` from config_flow and `_find_huawei_battery_device` from config_flow. For each default entity, check `hass.states.get(entity_id) is not None` to confirm existence. Returns `{"inverter_type": "huawei_sun2000", "detected": true, "sensors": {"battery_soc_sensor": "sensor.batteries_batterieladung", ...}, "huawei_device_id": "<device_id>"}` or `{"detected": false, "sensors": {}}` if no Huawei integration.

Each handler decorated with `@websocket_api.websocket_command(...)` and `@websocket_api.async_response`. Use `voluptuous` for schemas.

Export a single function:
```python
def async_register_websocket_commands(hass):
    websocket_api.async_register_command(hass, ws_get_config)
    websocket_api.async_register_command(hass, ws_save_config)
    websocket_api.async_register_command(hass, ws_check_prerequisites)
    websocket_api.async_register_command(hass, ws_detect_sensors)
```

Import pattern: `from homeassistant.components import websocket_api` (direct import, not via `hass.components`).
